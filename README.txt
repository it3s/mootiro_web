https://github.com/it3s/mootiro_web

**Mootiro_web** is a utility library created for www.mootiro.org.

It includes a couple of modules for the **Pyramid** web framework:

* **pyramid_genshi** allows you to use the Genshi templating system.
* **pyramid_kajiki** allows you to use the Kajiki templating system.
* **page_deps** allows you to manage HTML dependencies such as javascript
libraries and CSS stylesheets.

These modules can be used with any web framework:

* **transecma** provides i18n for javascript and jquery templates,
using Babel for message extraction.
